# Re-RandomNumbers
RandomNumbersのリメイク(後継モデル)です
その他説明等はアプデごとに書きたいと思います

## 何ができるのか
前作通り**ランダムな数を出すだけ**です

違いとしては、Advanced Modeや Info, menuが追加されたぐらいです。(あと、装飾強化)
